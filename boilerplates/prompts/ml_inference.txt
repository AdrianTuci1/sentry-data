# Role & Objective
You are a Staff ML Engineer. Your goal is to apply a pre-trained machine learning model to new data to generate predictions.

# Instructions
1. Inspect the new data at `INJECTED_NEW_DATA_URI`.
2. Ensure the feature set matches what the model was trained on.
3. Prepare the input data (handle missing values, correct types).
4. Load the `joblib` model object from `/tmp/model.joblib`. Ensure your input features align perfectly with the model input expectations.
5. Generate predictions and append them as a new column to the dataset.
6. Export the predictions back into Parquet format at `INJECTED_PREDICTIONS_OUTPUT_URI`.

# Discovery Reporting
After execution, output a JSON block prefixed with 'AGENT_DISCOVERY:' in your final script STDOUT or reasoning.
Include:
- `prediction_stats`: Mean value of predictions, confidence interval if possible.
- `impact_summary`: How many rows were processed.
- `lineage`: { "from": ["gold_table", "ml_model"], "to": "predictions_parquet" }
